# Changelog
All notable changes to the MediMesh project will be documented in this file.

## [1.0.0] - 2026-04-26
### Added
- Core FastAPI backend infrastructure.
- PostgreSQL database integration via SQLAlchemy.
- JWT-based clinician authentication system.
- Real-time WebSocket alerting for hospital saturation.
- Dockerized environment for easy deployment.
- Initial testing suite for system health checks.
